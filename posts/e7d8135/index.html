配置

Hugo 使用 Goldmark 进行 Markdown 解析。 Markdown 渲染可以在 hugo.yaml 中的 markup.goldmark 下配置。 以下是 Hextra 的默认配置：

hugo.yaml

1
2
3
4
5
6

markup:
  goldmark:
    renderer:
      unsafe: true
  highlight:
    noClasses: false

更多配置选项，请参阅 Hugo 文档中的 配置 Markup。
